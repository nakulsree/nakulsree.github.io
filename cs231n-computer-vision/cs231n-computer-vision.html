I

Why I Did This

After forecasting and optimization work, I wanted the same depth in vision. CS231n is the canonical path: it starts with classic image classification and ends with modern representation learning.

I implemented every section between the START CODE HERE and END CODE HERE markers myself. Stanford provides the scaffolding; the model logic, gradients, and training loops are handwritten.

◆
A note on framing: The assignments and problem structure belong to Stanford. This page documents my independent implementation of the mathematical core and my synthesis of the concepts across the full sequence.
What CS231n covers
End-to-end vision: image classification pipelines, CNNs, optimization and regularization, sequence models for image captioning, transformers, self-supervised learning, and diffusion-based image generation.
Why it matters
It connects representation learning, scalable optimization, and real-world vision datasets. This sequence forces you to derive gradients, implement backprop, and then scale up using PyTorch on real data.
II

Curriculum Highlights

Three assignments that progress from classic classifiers to modern generative and self-supervised learning.

Assignment 1 — Image Classification

Built the full image classification pipeline, implemented kNN and softmax classifiers, a two-layer network, and higher-level representations including HOG and color histograms.

kNN classifier Softmax classifier Two-layer neural net Feature engineering Fully connected nets
Assignment 2 — Deep Nets and CNNs

Implemented batch norm and dropout, constructed convolutional layers, trained CNNs in PyTorch, and built RNN-based image captioning models.

Batch normalization Dropout Convolutional networks PyTorch on CIFAR-10 RNN captioning
Assignment 3 — Modern Vision Models

Built transformer-based captioning, experimented with self-supervised learning, implemented DDPM diffusion models, and explored CLIP and DINO for representation learning.

Transformer captioning Self-supervised learning DDPM diffusion CLIP DINO
